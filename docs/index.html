Meridian Mail

Мессенджер на открытых протоколах

Переписка через IMAP / SMTP с UX мессенджера. Работает с Gmail, Yandex и любым своим сервером. Никаких серверов Meridian Mail — только клиент, который вы устанавливаете. Ваши письма остаются там, где были.

Что умеет

  • Диалоги, а не папка входящих
    Письма группируются по собеседнику. Чат-интерфейс вместо списка тем.

  • Любой IMAP/SMTP сервер
    Gmail, Yandex — через пароль приложения. Свой сервер — через ручные настройки.

  • Push без опроса
    IMAP IDLE — новые сообщения появляются мгновенно без постоянного polling.

  • Локальное хранение
    Все данные в SQLite на вашем устройстве. Никакого облака, никаких посредников.

  • macOS / Windows / Linux
    Нативный десктоп на Tauri + Rust. Лёгкий бинарь, системные шрифты.

Зачем

Telegram блокируют. WhatsApp принадлежит Meta. Slack — подписка у вендора. Любой мессенджер — это чей-то сервер в чьей-то юрисдикции.

Email — федеративный протокол уровня интернет-инфраструктуры. Нет единой точки отказа. Можно поднять свой сервер за 20 минут. Протоколу 40+ лет, он пережил всё. Заблокировать email = заблокировать интернет.

Meridian Mail берёт этот стек и даёт ему UX мессенджера.
